## Develop

git clone https://github.com/bburns/psyche-react-expo
cd psyche-react-expo
npm install
npm start

will open browser at http://localhost:19006

## Deploy

    npm run build
    npm run deploy

test that link, then

    npm run deploy:prod

site will be at https://psyche-react-expo.netlify.app/
