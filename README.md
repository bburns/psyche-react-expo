# Psyche

An open-source mental health app.

## About

## Develop

    git clone https://github.com/bburns/psyche-react-expo
    cd psyche-react-expo
    npm install
    npm run web

Browser will open at http://localhost:19006

## Deploy

    npm run build
    npm run deploy

Test that link, then

    npm run deploy:prod

Site will be at https://psyche-react-expo.netlify.app/
